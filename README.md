## 🚀 Como Usar

### Desenvolvimento
```bash
npm install
npm run dev
```

### Produção
```bash
npm run build
```

### Personalização

#### Alterar Link de Pagamento
Edite a linha 6 do arquivo `src/App.tsx`:
```typescript
const paymentLink = "https://pay.hotmart.com/W105482080M";
```

#### Alterar Preço
Busque por `R$ 29,90` no arquivo e substitua pelo valor desejado.

#### Personalizar Depoimentos
Edite a seção "Depoimentos" no arquivo `src/App.tsx` (linha ~540).
